# Saito-Menu

The ```saito-menu``` is class is used to create menu lists 


Each List type can be placed inside the saito-list container:

```1: saito-list-user```
```
                   <div class="saito-menu  ">
                        <ul>
                            <li>

                                <span> BlackJack </span>
                            </li>
                            <li>

                                <span> Chess</span>
                            </li>
                            <li>
                                <span> Red Imperium</span>
                            </li>
                            <li>
                                <span> Poker</span>
                            </li>
                            <li>
                                <span> Settlers of Saitoa</span>
                            </li>
                            <li>
                                <span> Solitrio</span>
                            </li>
                        </ul>
                    </div>
````
